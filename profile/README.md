## 📋 Overview

<a href="#perplexityAi">Perplexity AI</a> is an AI-powered answer engine that combines real-time web search with large language model synthesis — providing answers to questions with cited sources rather than responses generated purely from training data, addressing the core limitation of standard AI assistants for research and current information needs.

The <a href="#perplexityAi">web-grounded answers with citations</a> distinguish Perplexity from standard AI chatbots — every response references the web sources used to generate it, with numbered citations linking to the original pages. Users can verify the information, read the full source, and assess the source quality rather than receiving AI-generated text with no provenance. This citation model makes Perplexity appropriate for research tasks where source verification matters, and for current information that AI training data doesn't include. The <a href="#perplexityAi">real-time web search integration</a> means Perplexity's answers reflect current information — news from today, product releases from this week, recent research publications, and any other information that post-dates AI model training cutoffs. Standard AI assistants cannot answer questions about recent events reliably; Perplexity's search integration addresses this directly.

The <a href="#perplexityAi">conversational follow-up capability</a> continues research threads across multiple questions — asking a follow-up to a previous answer maintains the context of the earlier exchange, enabling iterative research that builds understanding progressively rather than treating each question as isolated. Complex research topics benefit from this conversational approach where each question builds on the previous answers. The <a href="#perplexityAi">focus modes</a> direct searches to specific content types — Academic mode searches peer-reviewed papers and academic sources, Writing mode assists with composition, YouTube mode finds video content, Reddit mode searches community discussions, and other modes target specific source types for queries where general web search would return less useful results.

<a href="#perplexityAi">Multiple AI model access</a> on Perplexity Pro provides access to different underlying models — switching between Claude, GPT-4, and other models for the same query to compare responses or use the model best suited to a specific task type. <a href="#perplexityAi">Collections for research organization</a> save searches and threads into named collections — building reference libraries of research on specific topics that persist across sessions for ongoing research projects.

<a href="#perplexityAi">Mac native application</a> provides a dedicated desktop experience beyond the web interface — keyboard shortcuts, system integration, and a window that coexists with other Mac applications for research alongside work rather than requiring browser tab management. <a href="#perplexityAi">Image understanding capability</a> accepts image uploads for analysis and questions — identifying objects, reading text in images, analyzing charts and diagrams, and answering questions about visual content.

## ❓ FAQ

<details>
<summary>What makes Perplexity AI different from ChatGPT?</summary>

Perplexity searches the web in real time and cites sources for every response. ChatGPT responds from training data without real-time search (unless using specific plugins). Perplexity is better for current information and research requiring source verification; ChatGPT has broader conversational and creative capabilities.

</details>

<details>
<summary>Does Perplexity AI cite its sources?</summary>

Yes. Every Perplexity response includes numbered citations linking to the web sources used to generate the answer.

</details>

<details>
<summary>Does Perplexity AI have real-time information?</summary>

Yes. Perplexity searches the web for each query, providing current information beyond AI model training data cutoffs.

</details>

<details>
<summary>What are focus modes in Perplexity?</summary>

Focus modes direct searches to specific content types — Academic for peer-reviewed papers, YouTube for video, Reddit for community discussions, and others. Each mode filters search results to the most relevant source type for the query.

</details>

<details>
<summary>Is there a free version of Perplexity AI?</summary>

Yes. Perplexity has a free tier with core search and answer capabilities. Pro adds higher limits, multiple AI model access, and additional features.

</details>

<details>
<summary>Does Perplexity AI have a Mac app?</summary>

Yes. A native Mac application provides a dedicated desktop experience with system integration and keyboard shortcuts.

</details>

<details>
<summary>Can I save research in Perplexity?</summary>

Yes. Collections save search threads into named groups for organizing ongoing research projects.

</details>

<details>
<summary>Does Perplexity support image input?</summary>

Yes. Images upload for analysis — identifying content, reading text, and answering questions about visual information.

</details>

<details>
<summary>What AI models does Perplexity Pro provide access to?</summary>

Perplexity Pro provides access to multiple models including Claude and GPT-4 in addition to Perplexity's own models. Check current Perplexity documentation for the available model list.

</details>
